## How to Run

**Option A — Direct open:**
1. Download or clone this repository
2. Double-click `index.html`
3. Done

**Option B — VS Code Live Server:**
1. Open the folder in VS Code
2. Right-click `index.html` → Open with Live Server

**Option C — Clone and open:**
```bash
https://github.com/Kirubelyitagesu/pathfinding_finder.git
cd pathfinding-visualizer
# open index.html in your browser
```

---

## Features

- Click and drag wall drawing
- Moveable start and end points
- Random maze generator
- Algorithm description card (updates when you switch algorithm)
- Speed slider (1 = slow and educational, 5 = instant)
- Stats panel: nodes explored · path length · time taken
- Color-coded legend
- Touch support for phones and tablets
